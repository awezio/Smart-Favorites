# Examples

## Paper Research Assistant

Upload PDF papers, ask questions through the knowledge base, then use `add_annotation` to keep notes on source documents.

## Public Wiki

Use `list_documents`, `search_knowledge`, and bookmark search to render a read-only public wiki from curated knowledge-base entries.

## Course Helper

Organize course PDFs and bookmarks by tag, query them with `query_semantic`, and generate study sessions from the highest scoring sources.

## Blog Writer

Search the knowledge base for source material, collect URLs and document references, then draft posts with citations in the writing app of choice.

## Research Agent

Give an AI agent an API key with `knowledge:read`, `documents:read`, and `stats:read`. The agent can call Smart Favorites tools to answer research questions grounded in the user's own data.
