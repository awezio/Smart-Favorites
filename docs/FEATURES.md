# Features

## Bookmarks

- Browser bookmark import and sync.
- Semantic search and keyword fallback.
- AI-generated descriptions.

## GitHub Stars

- Repository sync.
- Unified search with bookmarks.
- Metadata fields for owner, repo, language, stars, forks, and URL.

## Documents

- Upload PDF, DOCX, XLSX, TXT, Markdown, and HTML.
- Track processing status and errors.
- Store document metadata and chunks for retrieval.

## Square

- Community-style sharing surface.
- Posts, media storage, voting, and statistics APIs.

## Tools

- API-key protected HTTP tools.
- OpenAI, Claude, DeepSeek, and LangChain integration shapes.
- Permission-scoped tool discovery and execution.
